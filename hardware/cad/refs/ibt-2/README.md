# IBT-2 (BTS7960) motor driver reference model

High-power H-bridge driver (ordered ×3). Exported for electronics bay layout in Inventor.

| File | Format | Notes |
|------|--------|-------|
| `ibt-2.step` | STEP | **Module only** — use for chassis footprint (50×50×43 mm) |
| `ibt-2-wiring.step` | STEP | Module + signal wiring + mounting screws |

Board dims: 50 × 50 × 43 mm · ~66 g · 6–27 V · one per drive motor.
